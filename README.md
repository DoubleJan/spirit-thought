# Spirit Thought 灵思

## 一 项目概述

本项目是作者的个人主站项目，项目地址<spirithought.net>。本项目的主要技术栈

- 前端： `typescript` + `react`
- 后端：`nodejs`
- 数据库：`mongodb`
- 服务器：`linux` + `nginx`
- `文本编辑和渲染`：`markdown`

> 本项目的代码仅供个人学习或公益组织使用，禁止商用。
